# MEETranslation

Essa e um projeto para uma extensão que ira traduzir chamadas do MEET

# Como adicionar ao seu Chrome

<ol>

<li> baixar os arquivos</li>
<li> Abrir o chrome em gerenciador de extensão<li>
<li> Ativar o modo desenvolvedor </li>
<li> Com os arquivos em mão basta clicar em <b>Carregar sem compactação <b> </li>

</ol>

# Como funciona

<p> Para funcionar a extensao e necessario estar em uma reuniao e ativar as suas legendas (em portugues de preferencia), assim quando você e quem esteja junto com você na reunião falar ele pegara das legendas e traduzira para ingles e enviara no chat </p>
